# tempo-atual-nodejs
Exibe informações do tempo atual

```
node app.js "São Paulo"
```

 Saída:

```
 Campinas: 20º
 Agora: Chuva Fraca
 Humidade: 0.87
 Vento: 2.67
 Pressão: 1013.66
```

# API's utilizada

Coordenadas:

https://developers.google.com/maps/documentation/geocoding/intro?csw=1

Informações do tempo:

http://forecast.io/
